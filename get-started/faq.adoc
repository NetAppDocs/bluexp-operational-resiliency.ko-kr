== 운영 복구 서비스에 대한 액세스

* BlueXP 운영 복구 URL은 무엇입니까? *

URL의 경우 브라우저에 다음을 입력합니다. https://console.bluexp.netapp.com/["https://console.bluexp.netapp.com/"^]

* 운영 복구 서비스를 사용하려면 라이센스가 필요합니까? *

NetApp 라이센스 파일(NLF)은 필요하지 않습니다.
