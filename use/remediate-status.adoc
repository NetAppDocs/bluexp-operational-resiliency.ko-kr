= 개선 상태를 검토합니다
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/use/


[role="lead"]
언제라도 개선 상태를 확인할 수 있습니다. 실행, 완료 또는 실패 여부를 확인할 수 있습니다.

.단계
. BlueXP 왼쪽 탐색 창에서 * 상태 * > * 운영 복구 * > * 수정 상태 * 를 선택합니다.
+
수정 상태 페이지가 나타납니다.

. 문제의 세부 정보를 보려면 문제를 선택하여 확장합니다.
